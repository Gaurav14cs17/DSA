## Quick Reference

### Key Operations

| Operation | Complexity |
|-----------|:----------:|
| Build | O(n) |
| Point Update | O(log n) |
| Range Query | O(log n) |
| Range Update (Lazy) | O(log n) |

### Supported Operations

- **Sum:** Range sum queries
- **Min/Max:** Range minimum/maximum
- **GCD:** Range GCD queries
- **Count:** Range count with condition
